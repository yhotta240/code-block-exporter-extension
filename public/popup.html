説明

Code Block Exporter はウェブページ上のコードブロックを簡単に取り出せる Chrome 拡張機能です．コードのコピー，Markdown ブロック形式でのコピー，および選んだ拡張子でのダウンロードを簡単に行えます．

こんな人におすすめ

  • コードを繰り返し保存・整理する開発者
  • 技術記事やチュートリアルからコードを抜き出したい人
  • 学習用にコードを効率よく収集したい人

主な特徴

  • コードブロックをワンクリックでコピー／ダウンロード
  • 拡張子を選ぶだけでコードを簡単にダウンロード
  • GitHub，Qiita，Zennなどの定番技術サイトに対応
  • Markdown ブロック形式でそのままコピー可能
  • 言語を自動判定して最適な拡張子を付与

注意点

サイトごとの HTML 構造やブラウザ設定によって，トリガーやパネルの表示が期待どおりにならないことがあります．

サイトのスタイルに依存するため，パネルの表示が崩れることがあります．

  1. Chrome ウェブストアから Code Block Exporter をインストール
  2. ブラウザの右上のアイコンからポップアップを開く
  3. 「Code Block Exporter を有効にする」のトグルボタンをオンにする
  4. コードを含むページをブラウザで開くと，コードブロック上に右上に小さなトリガーが表示される．
  5. トリガーにマウスを合わせるとパネルが開く．パネル上でファイル名を編集し，表示されている拡張子ボタンをクリックしてダウンロードする．
  6. カスタム拡張子を使用する場合は，パネル右側の入力欄に拡張子を入力し，Enter キーを押す．

  • ページ内の pre / code 要素を検出し，コードブロックごとにトリガーを挿入する．
  • トリガーにマウスを合わせると，コードブロックの右上にポップオーバー型パネルを表示する．
  • ポップアップで設定した項目は chrome.storage.local に保存され，パネルの設定に即座に反映される．
  • ダウンロードは一時的な Blob URL を生成して行う．
  • 言語の検出はpre / code 要素，そのラッパー要素ののクラス名や属性などを参照して行う．

  • トリガー：コードブロック付近に表示される小さなボタン．マウスを合わせることでパネルが表示される．
  • パネル：コピーやダウンロードを行うための操作パネル．ファイル名の編集，拡張子の選択，コピー機能を備える．
  • エクスポート：本拡張機能において，コードのコピーおよびダウンロード操作を指す．
  • コードブロック：ページ内の pre / code 要素で構成されるコードの塊．
  • カスタム拡張子：ダウンロード時に使用する拡張子を任意に指定する機能．パネル右側の入力欄に拡張子を入力し，Enter キーを押すことで指定した拡張子でダウンロードされる．
  • 自動拡張子ボタン：コードの言語を判定し，自動的に選択された拡張子を示すボタン．パネル上にはオレンジ色で表示される．
  • 拡張子ボタン：ユーザがあらかじめ設定した拡張子を選択するためのボタン．パネル上には青色で表示される．
  • 有効／無効：拡張機能のオン／オフを切り替える設定である．無効にすると，トリガーおよびパネルは表示されない．
